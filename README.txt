Instructions:

Linux OS:

git clone https://github.com/Quinniboi10/voice-assistant.git
sudo apt-get install python3
cd voice-assistant
sudo pip3 install -r requirements.txt
python3 main.py

Mac OS:

git clone https://github.com/Quinniboi10/voice-assistant.git
comming soon...

Windows OS:

go to https://github.com/Quinniboi10/voice-assistant/archive/main.zip
unzip the folder
download and install anaconda here: https://www.anaconda.com/products/individual
launch spyder by pressing the windows key and typing spyder
go into the IPython console (in the bottom left)
type !pip install JarvisAI
then press green run arrow in the top right corner

Stay tuned for new commands and other cool upgrades!
